What is it all about?

If you're a Facebook user you're probably familiar with groups. People often use Facebook's groups to collaborate and share their thoughts. They also share files.

The problem with file sharing is that there are so many services to store files in the cloud, so people use all of them. Most often they paste a link to a file in a Facebook post.

We have the solution!

Facebook files app analyzes your groups and all posts, and looks for files. It then presents them in a nice way, so you can browse them as in standard file explorer. Nice, huh?
